# AdForge

Ads Management & Optimization Platform.

## Features

- **Multi-Platform**: Manage Meta, Google, TikTok, and X ads.
- **AI Creative**: Generate ad copy and landing pages with AI.
- **Auto-Optimization**: Automated rules to manage campaign spend.
- **MCP Enabled**: Integrated Model Context Protocol (MCP) for seamless AI agent automation.

## Quick Start

```bash
npm install
npm run build
npm start
```

## MCP Integration

AdForge provides an MCP server for integration with OpenClaw, Claude Code, and other AI agents.

**MCP Server Command**: `node mcp.js`

**Available Tools**:
- `adforge_list_campaigns`
- `adforge_get_analytics`
- `adforge_list_landing_pages`
- `adforge_list_creatives`

Refer to [SKILL.md](./SKILL.md) for detailed integration instructions.
